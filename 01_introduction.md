### What is System design?
- first step in the development of complex software
- blueprint for software
- you have to think about 
	- data structures
	- algorithms
	- architecture
	- integration points
- you have to think the factors like
	- extensibility
	- scalability
	- compatibility
	- maintainability
- you need to think about 
	- system behavior
	- workflow
	- components interact
